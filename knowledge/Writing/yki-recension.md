# YKI Formellt Brev: RECENSION — Ready Templates (80–100 ord)

---

## ⭐ POSITIV RECENSION TEMPLATE (Positive review)

```
[Platsens/Tjänstens namn]

[Platsens namn] är mycket bra! Jag [besökte/åt där/bodde där] [när/med vem].

Personalen var [adjektiv] och [adjektiv]. [Beskriv service].
Maten/Rummet/Produkten var [adjektiv]. [Beskriv detaljer].
Miljön/Atmosfären var [adjektiv]. [Beskriv stämningen].

Vi hade en toppenkväll/fantastisk upplevelse och kommer definitivt att [gå tillbaka/köpa igen].
Jag rekommenderar [namn] till alla som vill [ha god mat / bo bekvämt / få bra service].
```

---

## ⭐ NEGATIV RECENSION TEMPLATE (Negative review)

```
[Platsens/Tjänstens namn]

Jag hade en tråkig upplevelse på [platsens namn]. [Vad tyckte du överlag].

För det första var [personalen/rummet/maten] [negativt adjektiv]. [Förklara].
För det andra var [annat problem]. [Förklara].
Dessutom [ytterligare problem].

Sammanfattningsvis var min upplevelse inte bra.
Jag rekommenderar inte [namn] och kommer inte att [gå tillbaka/bo där] igen.
```

**Regler:**
- 80–100 ord
- Skriv platsens namn som rubrik
- Beskriv: personal + mat/rum/produkt + miljö/atmosfär + pris
- Avsluta med rekommendation (positiv eller negativ)
- Recensioner har INGEN hälsning (inget "Hej," eller "Med vänliga hälsningar")

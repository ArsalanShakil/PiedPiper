# YKI Formellt Brev: KLAGOMÅL — Ready Templates (80–100 ord)

---

## ⭐ KLAGOMÅL TEMPLATE (Complaint)

```
Hej,

Jag skriver till er eftersom [varför du skriver].

[Beskriv fakta: vad, när, var, ordernummer].
Tyvärr [beskriv problemet]. [Förklara hur det påverkade dig].

Jag skulle vilja [vad du vill ska hända: byta, ersättning, pengarna tillbaka].
Om det inte är möjligt vill jag [alternativ lösning].

Jag ser fram emot ert svar så snart som möjligt. Tack på förhand!

Med vänliga hälsningar,
[Namn]
Tel: [nummer]
E-post: [e-post]
```

**Regler:**
- 80–100 ord
- Hej, (aldrig Hej + namn)
- Använd NI / ER (inte du/dig) — du skriver till en organisation
- Sakligt, tydligt, fakta (datum, plats, ordernummer)
- Avsluta med vad du vill ska hända
- Med vänliga hälsningar + namn + kontaktuppgifter

# SNABBREFERENS — Klagomål-fraser

| Funktion | Fraser |
|----------|--------|
| Inledning | Jag skriver till er eftersom... / Jag kontaktar er angående... |
| Beskriv fakta | Jag köpte/beställde/bokade... den [datum]... / Ordernummer: #... |
| Problem | Tyvärr... / Det var mycket frustrerande. / Jag blev besviken. |
| Försökt kontakta | Jag försökte kontakta er kundtjänst men fick inte svar. |
| Vad du vill | Jag skulle vilja... / Om det inte är möjligt vill jag... |
| Bifoga bevis | Jag bifogar kvittot/bekräftelsen/detaljerna. |
| Avslut | Jag ser fram emot ert svar så snart som möjligt. Tack på förhand! |
| Hälsning | Med vänliga hälsningar, + Namn + Tel + E-post |

**Viktigt: NI / ER i klagomål**

| Informellt (vän) | Formellt (företag) |
|---|---|
| du | ni |
| dig | er |
| din/ditt | er/ert |
